# 撚二科生產戰情室 4.0 旗艦品質版技術手冊 (GEMINI4.0.md)

本文件定義 4.0 「旗艦品質版」的核心架構與運作邏輯，此版本整合了品質監控、智慧對接搜尋以及自動化美化報表。

## 1. 系統架構 (System Architecture)
1.  **`RealityLogAnalyzer` (多源數據引擎 - reality_analyzer.py)**：
    *   **品質集成**：整合《生產日報表》，追蹤各批號之「A 級率」與「定重率」。
    *   **智慧對接**：使用標準化批號（Standardized ID）串聯計畫、生產與庫存。
    *   **POY 消耗分析**：跨機台彙總 POY 需求，計算全場支撐天數。
2.  **`ExportEngine` (美化匯出引擎 - export_cleaned_plan.py)**：
    *   **邏輯清洗**：執行 ffill 補全機台與 POY，並剔除雜訊列（如機修、待排）。
    *   **旗艦美化**：自動調整欄寬、凍結首行、深藍專業表頭設計。
3.  **`DashboardRenderer` (渲染引擎 - generate_dashboard_v3.py)**：
    *   **整合 UI**：結合戰情室網頁與 Excel 匯出觸發。
    *   **智慧過濾**：整合工程師篩選、通用搜尋（機台/DTY）與 POY 追蹤。

## 2. 核心數據邏輯 (Core Logic)
*   **品質警示**：A 級率 < 95.0% 或 定重率 < 90.5% 標示為紅色警告。
*   **庫存對接 (Join Key)**：移除前綴（2F/2G）並處理結尾字母（如 2FA042S -> 2FA042），實現 A/AX 庫存智慧分攤。
*   **機台過濾**：嚴格剔除包含 V, S2, 庫存, 待排, 機台, M4, (CW) 以及 M01-08, S01-02 等非計畫生產數據。

## 3. UI/UX 規格 (v4.0+)
*   **導航增強**：
    *   **解除凍結**：頂部標題列改為相對定位，優化手機瀏覽空間。
    *   **回到頂部**：右下角新增平滑捲動按鈕 (▲)。
    *   **Excel 連結**：左上角「原始產銷(精簡版)」按鈕可直接開啟美化報表。
*   **智慧搜尋**：支援機台號碼與 DTY 批號之即時過濾，並與工程師分類連動。
*   **RWD 3.5**：針對 1200px 以下螢幕採用強化卡片佈局，所有標籤內容完整呈現。

## 4. 關鍵檔案清單
*   `reality_analyzer.py`：核心運算邏輯。
*   `generate_dashboard_v3.py`：網頁生成與 UI 控制。
*   `export_cleaned_plan.py`：Excel 美化匯出腳本。
*   `production_dashboard.html`：最終輸出成果。
