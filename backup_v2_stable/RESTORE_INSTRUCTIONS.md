# 系統還原教學 (撚二科產銷戰情室 2.0)

如果您在後續的架構大改中遇到無法修復的錯誤，請按照以下步驟進行還原。

## 方法一：自動還原 (推薦)
1.  在專案根目錄下，找到檔案：`還原穩定2.0版.bat`。
2.  **按兩下執行**該檔案。
3.  看到「系統已恢復至 2.0 穩定狀態」的字樣後，按下任意鍵關閉視窗。
4.  重新執行 `更新生產進度表.bat`，您的戰情室網頁將會變回最穩定的 2.0 版本。

## 方法二：手動還原 (當批次檔失效時)
如果自動還原無法執行，請手動將 `backup_v2_stable/` 資料夾內的檔案複製回根目錄，並重新命名：

| 備份檔案名稱 (在 backup 資料夾內) | 還原後的名稱 (放在根目錄) |
| :--- | :--- |
| `core_processor_v2.py` | `core_processor.py` |
| `generate_dashboard_v2.py` | `generate_dashboard.py` |
| `production_dashboard_v2.html` | `production_dashboard.html` |
| `GEMINI_v2.md` | `GEMINI.md` |

## 2.0 穩定版本功能回顧
這個版本包含以下所有關鍵功能：
*   **精確側邊判定**：支援 L 欄/M 欄作為 A/B 或 L/R 側的身分證。
*   **財務級彙總**：本月自產繳庫量 = (總庫存 H - 外購 H)。
*   **智慧對接**：2F42S2K 可自動對接到實產表。
*   **極致 RWD**：手機瀏覽不再空白，完美卡片顯示。
*   **備註顯示**：新增 DTY 表 C 欄備註顯示。
